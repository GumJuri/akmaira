# Akmaira Vercel Template

Статический шаблон для деплоя на поддомен Vercel `*.vercel.app`.

## Файлы

- `index.html` - страница с SEO-тегами и текстом про Akmaira.
- `styles.css` - легкий аниме-стиль, адаптив и анимации.
- `script.js` - мобильное меню, слайдер, плавное появление блоков и фоновые искры.
- `vercel.json` - базовая конфигурация для Vercel.
- `assets/` - папка для изображений и APK.

## Картинки для слайдера

Положите предоставленные изображения в папку `assets` с такими именами:

- `slide-1.jpg`
- `slide-2.jpg`
- `slide-3.jpg`

Если нужно больше слайдов, добавьте новые объекты в массив `sliderImages` в файле `script.js`.

## APK

Кнопка скачивания ведет на `./Akmaira.apk`. Если файл называется иначе, измените ссылку в `index.html` или переименуйте APK в `Akmaira.apk`.
